# Input data sources

## 1) Seed morphology phenotype data
Previously published supplementary data:
- Ahn et al. (2023) *Genome-wide association study of seed morphology traits in senegalese sorghum cultivars.*
- Supplementary Data S1 at the Plants article.

## 2) SNP genotype data
Public repository:
- DOI: https://doi.org/10.6084/m9.figshare.30877697
- Landing page: https://figshare.com/articles/dataset/_b_Genotypic_Data_SNPs_for_Sorghum_Mini_Core_Collection_and_Senegalese_Accessions_b_/30877697

## 3) Curated manuscript outputs
- Supplementary Data S1 associated with the revised spectral-manifold manuscript
- Existing GEMMA association outputs generated from the project

## 4) Raw hyperspectral imaging data
- Not bundled with this package
- Available from the corresponding author upon reasonable request

## 5) External binaries (not bundled)
- GEMMA: optional external execution for association scans if rebuilding from prepared inputs
- PLINK: not required for normal reviewer reruns of the shared downstream pipeline
